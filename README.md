# CCP Default Management — a training module

An open training module on how the default waterfalls of Australia's two central counterparties — **ASX Clear** and **ASX Clear (Futures)** — work in practice.

## What it covers

Seven modules plus an interactive scenario simulator:

- **Module 0** — How to use this
- **Module 1** — ASX's two CCPs (quick refresher)
- **Module 2** — Anatomy of a default waterfall
- **Module 3** — How a default actually unfolds (hours and days)
- **Module 4** — Interactive scenario simulator
- **Module 5** — Recovery tools deep dive (assessments, VMGH, partial tear-up, complete termination, replenishment)
- **Module 6** — Where does it fail?
- **Module 7** — Three workshop scenarios for discussion

Plus a glossary and a fully-cited Sources module pointing back to ASX rulebooks, RBA Financial Stability Standards assessments, CPMI-IOSCO PFMI, FSB guidance, and academic / industry sources.

## Audience

Readers who already understand what a CCP is and want sharper intuition about how default management actually unfolds — including the tools that come into play *after* the prefunded waterfall is exhausted.

## How to view

- **Online**: open `CCP-Default-Training-Public.html` in any modern browser.
- **GitHub Pages**: enable Pages in repo settings and serve from the root or `/docs`.
- The file is fully self-contained — no external CSS, JS, or fonts. It will work offline once downloaded.

Estimated runtime: ~90 minutes end-to-end. The simulator in Module 4 is the centrepiece — it walks an imposed loss through the actual rulebook tranches and shows where it breaks through into recovery and resolution.

## Currency of figures

All quantitative figures (default fund sizes, skin-in-the-game tranches, assessment caps) are as disclosed by ASX and the RBA in late 2025 / early 2026. ASX was consulting on a Dynamic Default Fund Framework as of July 2025 and may recalibrate tranche sizes. Always re-check against the current ASX disclosures and the most recent RBA Financial Stability Standards assessment before quoting figures externally.
